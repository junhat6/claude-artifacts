Claude Artifacts 保管庫の使い方

このページ自体が最初のアーティファクトです。スマホでこの URL が見えていれば、セットアップは成功しています。

公開のしかた

Claude Code のセッション内でアーティファクトを作ってもらったあと、次のコマンドを実行します:

/publish-artifact

直前に作られたアーティファクトの HTML がこのサイトに追加され、公開 URL が返ってきます。過去のファイルを指定して公開することもできます:

/publish-artifact /path/to/artifact.html

手動で公開する場合

cd ~/ghq/github.com/junhat6/claude-artifacts
python3 publish.py path/to/artifact.html
git add -A && git commit -m "Add artifact" && git push
⚠️ ここは公開インターネットです。 URL を知っていれば誰でも閲覧できます。仕事の内部情報・認証情報・個人情報を含むアーティファクトは公開しないでください。そういうものは claude.ai 上のアーティファクト（Claude アカウントで保護）のまま見るのが安全です。

しくみ

GitHub リポジトリ junhat6/claude-artifacts に HTML を貯め、GitHub Pages が main ブランチをそのまま配信しています。publish.py が断片 HTML を viewport 付きの完全な文書にラップし、トップの一覧ページを再生成します。push から反映まで最大1分ほどかかります。
